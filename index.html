Простой фреймворк для PHP

mini_framework представляет из себя простой MVC фреймворк для создание веб приложений.

<?php

/**
 * Данная функция вызывается до вызова 
 * опредленного действия в этом же контроле
 */
function actions_init () {
    
}

/**
 * Главное действие
 */
function action_index () {
    layout('index');
}

Особенности

  • Простой и документированный код

    Каждая функция в mini_framework документирована. Также весь код сам по себе прост. Если интересно как работает одна функция, Вам не составит труда разобратся в том как она работает.

  • Легкий

    Файлы mini_framework весят всего 36 КБ и mini_framework состоит из 14 файлов. Если эти файлы положить в zip архив, размер архива будет 13 КБ.

    К тому же, необязательно использовать все компоненты.

  • Вау как быстро!

    Из за того что mini_framework написан с использованием процедурного/функционального подхода, ваше приложение будет работать быстрее чем на ООП фреймворках. Единственное исключение - это фреймворки которые написаны как расширения для PHP на Си, возможно будут работать быстрее.

    В любом случае, выбирайте то что нравится или подходит.
